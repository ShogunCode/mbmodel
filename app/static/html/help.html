Getting Started

File Types: Only CSV files and TXT files are permitted for upload.


Preparation of the Data

Ensure your data is in the correct format, formatted from the Infinium Human Methylation 450K BeadChip readings. An example can be found here: Click Here!


Upload Process

On the left-hand section, locate and click the "Choose File" button to select your file.

Once you've selected the file, checks will be run on it.

After the checks are completed successfully, you'll be permitted to proceed with the upload.

Upon successful upload to the server, you will receive a confirmation message indicating success.


Processing of the Data

Click the newly appeared "Process Data" button to initiate the analysis.

The server will commence dimensionality reduction on the uploaded file.

Subsequently, the resulting data will be processed through the SVM classification model.


View Results

Once the analysis is complete, the results will be automatically displayed without leaving the page.

A box plot illustrating the range of confidence scores.

An interactive classification table showcasing the classification results.

A download button below the table, enabling you to download the data to your computer.

Additionally, you will be able to view a pie chart displaying the distribution of classifications derived from the user-uploaded data.
